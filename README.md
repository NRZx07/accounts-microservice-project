# Accounts Microservice Project

[![CI Build](https://github.com/NRZx07/accounts-microservice-project/actions/workflows/ci-build.yaml/badge.svg)](https://github.com/NRZx07/accounts-microservice-project/actions/workflows/ci-build.yaml)

## Project Overview
This project implements a robust microservice to manage account data, featuring full CRUD (Create, Read, Update, Delete) functionality, REST API endpoints, and automated CI/CD pipelines.

## Features
- RESTful API for Account Management
- Automated testing with `nosetests` and `flake8`
- Security integration with `Flask-Talisman` and CORS
- Containerized deployment using Docker and Kubernetes
